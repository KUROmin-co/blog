# Automated Profit Blog Configuration (Zero-Cost Setup)

## 1. Basic Info
- **Topic:** Global AI & Tech Trends (Curated & Summarized)
- **Target Audience:** Tech Enthusiasts, Developers, Early Adopters (Global/US-centric)
- **Language:** **English (US)**
- **Tone:** Professional yet accessible, Insight-driven, "Why this matters"

## 2. Content Sources (Free)
- TechCrunch AI
- Hacker News
- Twitter (X) AI Influencers
- Product Hunt / Reddit r/ArtificialInteligence

## 3. Monetization Strategy (Zero Initial Cost)
- **Phase 1 (Traffic):** SEO-optimized content targeting low-competition keywords
- **Phase 2 (Affiliate):** Referral links for AI tools (Jasper, Copy.ai, Notion AI, etc.)
- **Phase 3 (Newsletter):** Substack/Beehiiv (Free tier) subscription funnel

## 4. Automation Routine (Daily)
1. `web_search` for "Top AI news today"
2. Select top 3 articles
3. Summarize & Synthesize in English (Native-level quality)
4. Generate Markdown file in `_posts`
5. (Future) Auto-deploy to GitHub Pages
